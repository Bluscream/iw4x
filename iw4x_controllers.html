IW4x CONTROLLER GUIDE

An in-depth guide on how to setup controllers for IW4x.

Xbox

IW4x should support xbox controllers out of the box. If it does not work, make sure "Gamepad enabled" is set to "Yes" in the game options.

Playstation

  1. Download DS4Windows, make sure you pick DS4Windows_xxx_x64.zip
  2. Install/Extract it to a safe place and open it.
  3. It should detect your controller, but if it does not, you may need to install the driver.
  4. Check if gamepad works in IW4x. If there is still problems, you can join the Discord and ask for support in the #iw4x-support channel.

Common Issues that can prevent your Controller from working:

  1. Ensure that you are on the latest version of IW4x (you can check in the bottom right on the main menu for the build number). If you are not, you will need to update/reinstall.
  2. Disable Steam's built-in Controller configuration support or close Steam entirely.
  3. When using a PS Controller via DS4Windows make sure "Xbox Emulation" is enabled in the DS4Windows options
