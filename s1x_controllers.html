S1x CONTROLLER GUIDE

An in-depth guide on how to setup controllers for S1x.

Xbox

S1x should support xbox controllers out of the box. If it does not work, make sure "Gamepad Support" is enabled in settings.

Playstation

  1. Download DS4Windows, make sure you pick DS4Windows_xxx_x64.zip
  2. Install/Extract it to a safe place and open it.
  3. It should detect your controller, but if it does not, you may need to install the driver.
  4. Check if gamepad works in S1x. If there is still problems, you can join the Discord and ask for support in the #s1x-support channel.
